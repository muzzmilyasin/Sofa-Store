# Sofa - Furniture Website

A modern, responsive furniture website built with Bootstrap 5, featuring a clean design for showcasing furniture products and services.

## Features

- **Responsive Design**: Mobile-first approach with Bootstrap 5
- **Modern UI**: Clean and professional interface
- **Product Showcase**: Dedicated pages for products and services
- **Contact Forms**: Easy customer communication
- **Blog Section**: Content management for furniture tips and ideas

## Pages

- **Home** (`index.html`) - Landing page with hero section and featured products
- **Shop** (`shop.html`) - Product catalog and shopping interface
- **About** (`about.html`) - Company information and team
- **Services** (`services.html`) - Service offerings and features
- **Blog** (`blog.html`) - Articles and furniture tips
- **Contact** (`contact.html`) - Contact form and information
- **Cart** (`cart.html`) - Shopping cart functionality
- **Checkout** (`checkout.html`) - Order processing
- **Thank You** (`thankyou.html`) - Order confirmation

## Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- Font Awesome Icons

## Getting Started

1. Clone the repository
2. Open `index.html` in your browser
3. Navigate through the website using the navigation menu
